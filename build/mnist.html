How does it work?

Image classifiers often use convolution layers. To put it simply, these look at a neighbourhood of inputs and "summarize" what they see into a higher-dimensional representation. By chaining multiple after each other, you can learn high level features that then help classification. For example, the first layer might learn to detect lines, edges or other local features, while the later ones will put this information together and learn representations of shapes, such as where a cat's eyes are, etc.

The softmax function

Classifiers like this one tend to use a softmax function as output. You can think of it a probability distribution; it gives a value for each label, and then normalizes them so they sum up to one. You then typically take the label with the highest value and consider that your classification. You can also use that value for reasoning about the AI's confidence for example.
